# CV Interactif - Antoine JACROT

4 versions web interactives d'un CV, parce que les PDF c'est ennuyeux.

## Versions

| Version | Style | Description |
|---------|-------|-------------|
| V1 | Terminal | Style ligne de commande, easter eggs |
| V2 | RPG | Livre dont vous êtes le héros |
| V3 | Blackjack | Débloquez le CV en gagnant |
| V4 | Matrix | Pour ceux qui ont pris la pilule rouge |

## Voir en ligne
